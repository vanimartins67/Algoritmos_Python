# CRUD Usuário - Python + MySQL

Projeto simples para praticar CRUD (Create, Read, Update, Delete).

## Tecnologias
- Python
- MySQL

## Como usar
1. Criar o banco com o arquivo SQL
2. Instalar dependência:
   pip install mysql-connector-python
3. Rodar o arquivo main.py

## Funções
- create_user
- read_users
- update_user
- delete_user
